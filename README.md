# Productivity Time Tracker ⏱️

A Personal Web App for tracking productive hours with smart daily targets.

## Features

- ✅ Track productive hours daily
- ✅ Set and monitor daily/weekly/monthly targets
- ✅ PWA - Installable on mobile & desktop
- ✅ Cloud sync (requires authentication)
- ✅ Local storage mode (no account needed)
- ✅ Offline support
- ✅ Responsive design

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Shadcn UI
- NextAuth.js
- Google OAuth
- next-pwa

## Installation
